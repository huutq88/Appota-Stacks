## 🛠️ Cài đặt

### macOS
1. Tải file `.dmg`
2. Mở file → **kéo AppotaStacks vào thư mục Applications**
3. Mở AppotaStacks từ Launchpad hoặc Applications

> Nếu bị chặn bởi Gatekeeper: **System Settings → Privacy & Security → Open Anyway**

### Windows
1. Tải file `.exe`
2. Chạy installer → **Next → Install**
3. Tìm AppotaStacks trong Start Menu
